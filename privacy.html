Privacy Policy

Last updated: July 3, 2026

Overview

Windows Security Threat Scanner (WSTS) is a local, offline security tool. It makes no outbound network connections and transmits nothing from your machine. This policy explains what the website (wsts.spatcyber.com) does with the limited information that any web server necessarily receives.

The WSTS application

The WSTS executable runs entirely on your device:

  • It reads local Windows artifacts (Prefetch, Event Logs, Registry hives, Credential Manager files) only to display results in your browser.
  • It opens a local web server at http://127.0.0.1:5900 — accessible only on your machine.
  • No scan results, file contents, paths, or telemetry are ever uploaded.
  • No account, registration, or email address is required to run WSTS.
  • No cookies, tracking pixels, or analytics are embedded in the dashboard.

This website (wsts.spatcyber.com)

Information we receive automatically

When you visit this site your browser sends a standard HTTP request that includes:

  • Your IP address
  • Browser type and version (User-Agent header)
  • The page or file requested and the HTTP status returned
  • Referring URL (if you followed a link)
  • Date and time of the request

These are recorded in standard nginx access logs on our server and retained for up to 30 days for security, abuse prevention, and operational purposes. We do not sell, share, or otherwise disclose them to third parties except as required by law.

Cookies & tracking

This website sets no cookies and loads no third-party scripts or tracking pixels. There is no analytics code on this site.

Download analytics

We log download requests (file name, IP address, timestamp) from our server’s access log to detect abuse and count releases. This data is not linked to any personal profile.

Children’s privacy

This site is not directed at children under 13. We do not knowingly collect personal information from children.

Changes to this policy

If we make material changes we will update the “Last updated” date above. Continued use of the site after a change constitutes acceptance of the revised policy.
